## Брейкпоинты

```css
/* Mobile */
@media (max-width: 767px) { }

/* Tablet */
@media (min-width: 768px) and (max-width: 1023px) { }

/* Desktop */
@media (min-width: 1024px) { }

/* Tablet Landscape */
@media (min-width: 768px) and (max-width: 1023px) and (orientation: landscape) { }
```

## Возможные проблемы и решения

### Проблема: Текст слишком мелкий на телефоне
**Решение:** Увеличь font-size в секции Mobile в responsive.css

### Проблема: Кнопки сложно нажимать
**Решение:** Увеличь min-height и padding для кнопок в responsive.css

### Проблема: Изображения слишком большие
**Решение:** Добавь max-width: 100% и height: auto

### Проблема: Горизонтальная прокрутка
**Решение:** Проверь overflow-x: hidden на body и html

## Дополнительные улучшения (опционально)

### 1. Оптимизация изображений
```html
<!-- Используй srcset для разных размеров экрана -->
<img 
  src="image-mobile.jpg" 
  srcset="image-mobile.jpg 480w, 
          image-tablet.jpg 768w, 
          image-desktop.jpg 1200w"
  sizes="(max-width: 767px) 100vw, 
         (max-width: 1023px) 50vw, 
         33vw"
  alt="Description"
/>
```

### 2. Lazy loading для изображений
```html
<img src="image.jpg" loading="lazy" alt="Description" />
```

### 3. Preload критических ресурсов
```html
<link rel="preload" href="css/style.css" as="style" />
<link rel="preload" href="Materials/erasebg-transformed.png" as="image" />
```
