# Адаптивний дизайн для ASL Company - Короткий гайд

## Що зроблено ✅

Створено файл `css/responsive.css` з повною адаптацією сайту для:
- 📱 **Телефонів** (320px - 767px)
- 📱 **Планшетів** (768px - 1023px)  
- 💻 **Десктопів** (1024px+)

## Основні покращення

### Телефони:
- Логотип 60px (замість 120px)
- Великі кнопки для зручного натискання (44px+)
- Текст зручного розміру
- Мобільне меню працює ідеально
- Карточки в 1 колонку

### Планшети:
- Логотип 80px
- Карточки в 2 колонки
- Оптимізовані розміри тексту
- Підтримка горизонтальної орієнтації

## Як перевірити

### Швидкий тест у браузері:
1. Відкрий сайт у Chrome
2. Натисни **F12**
3. Натисни **Ctrl+Shift+M**
4. Вибери пристрій (iPhone, iPad)
5. Перевір всі сторінки

### На реальному телефоні/планшеті:
1. Відкрий сайт
2. Перевір вертикальну та горизонтальну орієнтацію
3. Перевір, чи зручно натискати кнопки
4. Перевір, чи читабельний текст

## Що перевірити на кожній сторінці

✅ **Головна:**
- Логотип правильного розміру
- Меню відкривається
- Перемикач мови працює
- Карточки категорій виглядають добре
- Кнопки легко натискаються

✅ **Асортимент:**
- Карусель брендів працює
- Кнопки навігації видно
- Зображення завантажуються

✅ **Новини:**
- Фільтри працюють
- Карточки новин читабельні
- Текст не обрізається

✅ **Вакансії:**
- Карточки вакансій читабельні
- Списки форматовані правильно

✅ **Контакти:**
- Контактна інформація читабельна
- Іконки відображаються

## Можливі проблеми

### Текст занадто дрібний?
Відкрий `css/responsive.css`, знайди секцію Mobile і збільш `font-size`

### Кнопки важко натискати?
Збільш `min-height` та `padding` для кнопок у `css/responsive.css`

### Горизонтальна прокрутка?
Перевір `overflow-x: hidden` на body

## Додаткові покращення (опціонально)

### 1. Оптимізація зображень
Стисни зображення через:
- TinyPNG.com
- ImageOptim
- Squoosh.app

### 2. Lazy loading
Додай до зображень:
```html
<img src="image.jpg" loading="lazy" alt="Опис" />
```

### 3. Перевірка швидкості
- Google PageSpeed Insights: https://pagespeed.web.dev/
- Введи адресу свого сайту
- Подивись рекомендації
